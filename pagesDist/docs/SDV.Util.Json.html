SDV.Util. Json

Base namespace for SDV.js library JSON utils. All supporting JSON utils are accessible through it.

(static) getFromFile(options) → {object}

Makes a GET-request to address specified in 'filePath'-option, and receives JSON-object (in a case of success). In case of invalid data in file, data will be received as string instead of JSON-object.
Parameters:
Name Type Description
options object Request options.
Properties
Name Type Attributes Default Description
filePath string Path to target JSON-file (for most browsers it could be even cross domain URL).
async boolean <optional>
 true Flag: represents if request should be asynchronous or not.
timeout number <optional>
 0 Request timeout (Non negative integer value).
onSuccess function Callback-function that handles successful request completion.
onError function <optional>
 function(error, xhr) { console.error(error.message); } Callback-function that handles failed request.
Source:
  • sdv.js, line 1386
Returns:
Returns XMLHttpRequest-object, or ActiveXObject (for old IE versions), or XDomainRequest-object (for IE8/IE9 if given file path is cross domain URL).
Type
object
Example
// If server contains file 'myJsonFile.json' in the same path as the current page, and that file contains following data:
// { "property1": 1 }, then onSuccess callback-function will receive that data and log it to console.
// In the case of fail, error message will be logged to browser console.
SDV.Util.Json.getFromFile({ filePath: 'myJsonFile.json', onSuccess: function(data, xhr) { console.log(data); } });

  • sdv.js, line 1303
Throws:
Throws a SyntaxError exception if the given string contains invalid JSON.
Type
SyntaxError
Returns:
Returns JSON-object, if given string contains valid stringed JSON.
Type
object
Example
SDV.Util.Json.parse(); // Will throw SyntaxError.
SDV.Util.Json.parse(''); // Will throw SyntaxError.

// Will throw SyntaxError, because in valid stringed JSON properties names must be enclosed in double quotes.
SDV.Util.Json.parse('{ property1: 1 }');

SDV.Util.Json.parse('{ \"property1\": 1 }'); // Will return following JSON-object: { property1: 1 }.

(static) stringify(value) → {string}

Transforms given JSON-object into string, contains stringed representation of given object. Note, that all symbol-keyed properties will be completely ignored.
Parameters:
Name Type Description
value object JSON-object being converted to a string.
Source:
  • sdv.js, line 1327
Returns:
Returns string, contains stringed representation of given JSON-object.
Type
string
Example
SDV.Util.Json.stringify({});                  // '{}'
SDV.Util.Json.stringify(true);                // 'true'
SDV.Util.Json.stringify('foo');               // '"foo"'
SDV.Util.Json.stringify([1, 'false', false]); // '[1,"false",false]'
SDV.Util.Json.stringify({ x: 5 });            // '{"x":5}'
SDV.Util.Json.stringify({ x: 5, y: 6 });      // '{"x":5,"y":6}' or '{"y":6,"x":5}'

// Symbol-keyed properties will be completely ignored:
SDV.Util.Json.stringify({ x: undefined, y: Object, z: Symbol('') }); // '{}'
