SDV.Util. Xml

Base namespace for SDV.js library XML utils. All supporting XML utils are accessible through it.

(static) getFromFile(options) → {object}

Makes a GET-request to address specified in 'filePath'-option, and receives XML-document (in a case of success). In case of invalid data in file, data will be received as string instead of XML-document.
Parameters:
Name Type Description
options object Request options.
Properties
Name Type Attributes Default Description
filePath string Path to target XML-document (for most browsers it could be even cross domain URL).
async boolean <optional>
 true Flag: represents if request should be asynchronous or not.
timeout number <optional>
 0 Request timeout (Non negative integer value).
onSuccess function Callback-function that handles successful request completion.
onError function <optional>
 function(error, xhr) { console.error(error.message); } Callback-function that handles failed request.
Source:
  • sdv.js, line 1566
Returns:
Returns XMLHttpRequest-object, or ActiveXObject (for old IE versions), or XDomainRequest-object (for IE8/IE9 if given file path is cross domain URL).
Type
object
Example
// If server contains file 'myXmlFile.xml' in the same path as the current page, and that file contains following data:
// <node1>1</node1>, then onSuccess callback-function will receive that data and log it to console.
// In the case of fail, error message will be logged to browser console.
SDV.Util.Xml.getFromFile({ filePath: 'myXmlFile.xml', onSuccess: function(data, xhr) { console.log(data); } });

Checks whether a given object is an XML-document.
Parameters:
Name Type Description
xmlObject object Inspected object.
Source:
  • sdv.js, line 1425
Returns:
Returns true if inspected object is an XML-document, otherwise returns false.
Type
boolean
Example
SDV.Util.Xml.isXml(); // false.
SDV.Util.Xml.isXml(null); // false.
SDV.Util.Xml.isXml('This is a string'); // false.
SDV.Util.Xml.isXml(1); // false.
SDV.Util.Xml.isXml([1]); // false.
SDV.Util.Xml.isXml(function(){return 1;}); // false.
SDV.Util.Xml.isXml({one: 1}); // true.
SDV.Util.Xml.isXml(SDV.Util.Xml.parse('<node1>1</node1>')); // true.

Transforms given XML-document into string, contains stringed representation of given XML.

Name Type Description
xmlObject document XML-document being converted to a string.

Throws:
  • Throws a TypeError exception if given object is not a document at all, if it is, for example, a number, or a string.
    Type
    TypeError
  • Throws a SyntaxError exception if the given object is not valid XML-document.
    Type
    SyntaxError
  • Throws common Error exception if browser don't support any XML-serialization facilities.
    Type
    Error
Returns:
Returns string, contains stringed representation of given XML-document.
Type
string
Example
SDV.Util.Xml.stringify();                  // It will throw TypeError.
SDV.Util.Xml.stringify(true);                // It will throw TypeError.
SDV.Util.Xml.stringify('foo');               // It will throw TypeError.
SDV.Util.Xml.stringify([1, 'false', false]); // It will throw TypeError.
SDV.Util.Xml.stringify({ x: 5 });            // It will throw TypeError.

// It will throw SyntaxError, because '<node1>' isn't closed.
SDV.Util.Xml.stringify(SDV.Util.Xml.parse('<node1>1</blahblah>'));

// It will return '<node1>1</node1>'.
SDV.Util.Xml.stringify(SDV.Util.Xml.parse('<node1>1</node1>'));
