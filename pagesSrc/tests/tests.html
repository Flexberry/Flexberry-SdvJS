Used technologies

Unit testing is organized using the following technologies:
  • Mocha a feature-rich JavaScript test framework.
  • Chai a BDD / TDD assertion library that can be delightfully paired with any JavaScript testing framework.
  • Sinon a library contains test spies, stubs, and mocks, which can be used with any unit testing framework.
  • For better convenience, we use chai-plugin Sinon-Chai which provides a set of custom assertions for using the Sinon spy, stub, and mocking framework with the Chai assertion library.
